# V1 — Path of Light

## מטרות

- להציג בסיס תלת־ממדי ללא ספריות חיצוניות.
- שליטה קלאסית במקלדת, שלושה נתיבים ומכשולים נעים.
- מדדים: נקודות הישרדות, מד מהירות ו-High Score מקומי.

## מכניקה

- מכשולים נוצרים במרחק `-1400px` ומתקדמים עם `translate3d`.
- מהירות בסיסית: `320 units/s` עם האצה עדינה (`+6 units/s` לשנייה).
- התנגשות: חפיפה בנתיב + חלון Z בין `-40` ל-`60`.
- ריסט: מקש Enter מנקה אובייקטים ומאפס ניקוד.

## רעיונות להרחבה

- מד אנרגיה למגן/דש.
- רטט/פלאש עדין בעת התנגשות.
- דינמיקת מסלולים מתחלפים.
