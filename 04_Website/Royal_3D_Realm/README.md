# 👑 Royal 3D Realm — Browser Game Lab

שלושה אב-טיפוס תלת־ממדיים קלים להרצה בדפדפן, נבנו ב-HTML/CSS/JS בלבד וללא ספריות חיצוניות. כל גרסה מציעה שדרוג חווייתי על פני הקודמת וניתנת להרצה כעמוד עצמאי.

## 🗂️ תוכן התיקייה

- `STRATEGY_NOTES.md` — עקרונות עיצוב, יעדים ומפת דרכים.
- `version1_path_of_light/` — רץ מסלולים בסיסי עם שליטה במקלדת.
- `version2_crystal_guard/` — הוספת איסוף קריסטלים, מד אנרגיה ודיפרנציאל קושי.
- `version3_celestial_surge/` — מוסיף מד קומבו, שליטה בעכבר/מגע ואפקטים סביבתיים.

## 🚀 הרצה

פתח כל `index.html` ישירות בדפדפן, או שרת מקומי לדוגמה:

```bash
cd Royal_3D_Realm/version1_path_of_light
python -m http.server 8000
# או עבור גרסאות אחרות: החלף את שם התיקייה
```

גלוש אל `http://localhost:8000`.

## 🎮 מהירות התחלה

- ⌨️ **חצים / A D** — מעבר בין נתיבים
- 🛡️ **Space** — דש/מגן (מגירסה 2)
- 🖱️ **Mouse / Touch** — גרירה אופקית לשליטה (מגירסה 3)

## 🧭 עקרונות חוויית משחק

- תנועה חלקה עם תחושת עומק (CSS 3D + פרוג'קשן ידני).
- משוב חזותי ברור: צבעים מלכותיים, ספלאשים עבור פגיעות/איסוף.
- התאמה למקלדת, עכבר ומסך מגע ללא התקנות נוספות.

## 🧪 בדיקות ידניות מומלצות

- מעבר בין נתיבים מגיב ללא קפיצות.
- זיהוי התנגשות פועל גם במהירויות גבוהות.
- הדש במרחב זמן קצוב מגיב ומגן (גרסה 2+).
- קצב הקושי עולה בהדרגה (גרסה 2+) תוך שמירה על פריימים יציבים.
- שליטה בעכבר/מגע לא "בורחת" מגבולות המסלול (גרסה 3).
