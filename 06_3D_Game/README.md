# 👑 SparKing 3D Runner — משחק דפדפן תלת־מימדי

קובץ זה מרכז את הפרטים האסטרטגיים והיישומיים של משחק ריצה תלת־מימדי חדש המבוסס על אסתטיקת SparKing. כל גרסה נשמרת בתיקייה ייעודית עם קבצי HTML/CSS/JS נקיים ללא תלות בבילדרים.

## 🎯 חזון המשחק
- **חוויה מהירה וברורה**: ריצה במסלול מלכותי מלא מחסומים ואוצרות זהובים.
- **אסתטיקה עקבית**: שימוש בצבעי הזהב/כחול/שחור של SparKing, תאורה דרמטית וניצוצות זוהרים.
- **התקדמות בשלבים**: שלוש גרסאות ראשוניות, כל אחת מוסיפה שכבת עומק חדשה.

## 🧭 מבנה תיקיות
```
06_3D_Game/
├─ README.md                     # מסמך זה
├─ Version_01/                   # גרסת אב־טיפוס בסיסית, רצה ומכשולים
├─ Version_02/                   # מוסיפה אספנים, קושי דינמי והשהייה
└─ Version_03/                   # מוסיפה אויבים נעים, אפקטים ומכניקת פוקוס
```

## 🕹️ מקשי שליטה (כל הגרסאות)
- ⬅️ ➡️ / A D — מעבר בין נתיבים
- ⬆️ / W / רווח — קפיצה קצרה (חסימת התנגשות נמוכה)
- R — אתחול הסשן
- P — השהייה (מגרסה 02 ומעלה)
- Enter — התחלה מחדש לאחר Game Over

## 🚧 מצב הנוכחי של הגרסאות
- **Version_01** — מסלול תלת־מימדי בסיסי עם מחסומים, ניקוד מצטבר וסיום משחק בהתנגשות.
- **Version_02** — מוסיפה אספנים זהובים, מד אנרגיה לקפיצות, האטה/האצה דינמית והשהייה.
- **Version_03** — מוסיפה יריבים נעים לרוחב, התאמת זווית מצלמה לנתיב, פוקוס (Bullet Time) שמאט זמן ומתגמל צבירת נקודות.

## 📋 הוראות הפעלה
1. פתחו את אחת מתיקיות הגרסה (`Version_0X/`).
2. טעינת `index.html` בדפדפן תציג את המשחק ללא תלות בשרת.
3. רוצים להריץ לוקלית עם שרת? אפשר `python -m http.server 8000` בתיקייה הרלוונטית ולגלוש ל־`http://localhost:8000`.

## 🔮 צעדי המשך מוצעים
- הוספת שמירת ניקוד שיא ב־LocalStorage.
- אפקטי צליל ופס־קול דינמי לפי קושי.
- שילוב HUD מלכותי מתקדם עם אייקונים מהמאגר הגרפי.
- מפות מוכנות מראש בנוסף לספאון אקראי.
