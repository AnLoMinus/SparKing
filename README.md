## 🚀 התחלה מהירה

### 📥 התקנה

```bash
# שכפול המאגר
git clone https://github.com/AnLoMinus/SparKing.git

# מעבר לתיקייה
cd SparKing

# פתיחת האתר (אופציונלי)
open 04_Website/index.html
```

### 🎨 שימוש בתבניות הגרפיות

1. פתח את התבניות מתיקיית `02_Royal_Cards/`
2. השתמש בפלטת הצבעים מ־`GRAPHIC_GUIDE_LINE.md`
3. שמור על הסגנון המלכותי המוגדר
